==============================
CRM Timesheet Analytic Partner
==============================

This module modifies phone call timesheets. It sets "Other partner" as partner
from phone call.

Installation
============

This module is auto-installed by Odoo when *crm_timesheet*
and *analytic_partner_hr_timesheet* are installed.

Usage
=====

In phone calls tree, if you add an analytic account and the call has any
duration and a partner, the created timesheet activity will have
"Other partner" assigned.
